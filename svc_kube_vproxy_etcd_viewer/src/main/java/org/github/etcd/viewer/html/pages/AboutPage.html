About etcd viewer

Etcd viewer is open source licensed under Apache 2.0 license and it is hosted on github.

Please feel free to report an issue or share your ideas.

View documentation on github

Contributions are always welcomed.

Features

  • Etcd v2.0 rest API
  • Filesystem like key-value store navigation
  • Add/delete directories
  • Add/delete/update key-value pairs
  • Enable Time To Live on directories and values
  • Multi-registry support
  • Etcd cluster monitor
  • Leader auto detection
